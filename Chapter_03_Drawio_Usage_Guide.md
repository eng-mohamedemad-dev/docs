# تشغيل وتعديل رسومات Chapter 3 على Draw.io

الملف المطلوب:

`diagrams/chapter-03/CrimeLens_Chapter_03_Diagrams.drawio`

هذا ملف Draw.io أصلي متعدد الصفحات، وليس كودًا تحتاج إلى نسخه وتشغيله يدويًا.
كل صفحة داخل الملف تمثل رسمة مستقلة من رسومات Chapter 3.

## الطريقة الأولى: من داخل VS Code

1. تأكد أن إضافة **Draw.io Integration** مثبتة ومفعلة.
2. افتح ملف `CrimeLens_Chapter_03_Diagrams.drawio`.
3. إذا ظهر الملف كنص XML:
   - اضغط بزر الفأرة الأيمن على اسم الملف.
   - اختر **Open With...**.
   - اختر **Draw.io Editor**.
4. ستظهر أسماء صفحات الرسومات أسفل المحرر، مثل:
   - `01 - System Context`
   - `08 - Use Case Overview`
   - `14 - Sequence AI Detection to Incident`
5. اختر الصفحة المطلوبة، ثم عدّل العناصر بالسحب أو بالنقر المزدوج على النص.
6. استخدم `Ctrl + S` لحفظ التعديلات داخل ملف Draw.io نفسه.

## الطريقة الثانية: باستخدام موقع Draw.io

1. افتح: <https://app.diagrams.net/>
2. اختر **Device** عند سؤالك عن مكان حفظ الملفات.
3. اختر **Open Existing Diagram**.
4. افتح ملف `CrimeLens_Chapter_03_Diagrams.drawio`.
5. تنقل بين صفحات الرسومات من الألسنة الموجودة أسفل الشاشة.
6. بعد التعديل اختر **File → Save** أو `Ctrl + S`.
7. نزّل الملف المعدل واحتفظ به بدل المصدر القديم إذا كنت تعمل من المتصفح.

## أهم عمليات التعديل

- تحريك عنصر: اسحبه بالفأرة.
- تعديل النص: انقر على العنصر نقرتين.
- تغيير الحجم: اسحب نقاط التحكم الزرقاء حول العنصر.
- نسخ عنصر: `Ctrl + D` أو `Ctrl + C` ثم `Ctrl + V`.
- حذف عنصر: حدده ثم اضغط `Delete`.
- رسم علاقة: اسحب من أحد أسهم الاتصال التي تظهر حول العنصر.
- تعديل نوع السهم: حدد الخط ثم استخدم لوحة **Style**.
- ترتيب العناصر: **Arrange → Align** و **Arrange → Distribute**.
- نقل عنصر أمام أو خلف عنصر آخر: **Arrange → To Front / To Back**.
- التراجع وإعادة التنفيذ: `Ctrl + Z` و`Ctrl + Y`.
- التكبير والتصغير: `Ctrl` مع عجلة الفأرة.

لا تغيّر دلالة الأسهم أو رموز الرسم لمجرد تحسين الشكل. علاقات `include`
و`extend`، رسائل Sequence Diagram، حراس Activity Diagram، وتعدد العلاقات
في Class Diagram لها معانٍ هندسية محددة.

## تصدير كل رسمة إلى PNG

1. افتح صفحة الرسمة المطلوبة من الشريط السفلي.
2. اختر **File → Export As → PNG**.
3. استخدم الإعدادات التالية:
   - **Zoom:** ‏`200%` على الأقل، و`300%` أفضل للنصوص الكثيرة.
   - **Border Width:** ‏`20`.
   - **Crop:** مفعّل.
   - **Transparent Background:** غير مفعّل.
   - صدّر الصفحة الحالية فقط. إذا ظهر خيار **All Pages** فلا تفعّله.
4. اضغط **Export**.
5. سمّ الصورة بالاسم المكتوب داخل Placeholder في Word. توجد القائمة الكاملة
   داخل `Chapter_03_Diagram_Requirements.md`.

## وضع الصورة داخل Word

1. افتح الصفحة التي تحتوي على Placeholder.
2. انقر داخل مربع الـPlaceholder.
3. من **Table Layout → Properties → Row** ألغِ خيار **Specify height** حتى لا
   تُقص الصورة.
4. احذف نص الـPlaceholder فقط، ولا تحذف عنوان الشكل الموجود تحته.
5. اختر **Insert → Pictures → This Device** وحدد صورة PNG.
6. اجعل **Wrap Text** مساويًا لـ **In Line with Text**.
7. وسّط الصورة واجعل عرضها قرابة `15.5–16 cm` مع الحفاظ على النسبة.
8. راجع وضوح النص في عرض الصفحة الكامل قبل الحفظ.

## ترتيب التصدير

ابدأ من الصفحة `01` حتى الصفحة `21`. اسم كل صفحة داخل Draw.io يقابله اسم
ملف محدد داخل `Chapter_03_Diagram_Requirements.md`، ولذلك لن تحتاج إلى تخمين
مكان أي رسمة داخل الكتاب.
